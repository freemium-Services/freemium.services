❓ Frequently Asked Questions

What is Freemium.Services?+
Freemium.Services is the world's largest verified directory of freemium, free, and open-source software tools. We curate, verify, and provide step-by-step self-hosting guides for 2,847+ tools across 47 categories. Every listing is manually verified to confirm the free tier or open-source license is genuine.
What does "freemium" mean in this context?+
Freemium tools are software products that offer a meaningful free tier — not just a 14-day trial. This includes fully open-source tools (self-hostable for free), tools with permanent free plans, and tools where the core functionality is free with optional paid upgrades.
How do I self-host these tools?+
Each tool page includes a dedicated self-hosting section with step-by-step instructions. Most include Docker Compose files, environment variable references, and cloud-specific guides for AWS, GCP, Hetzner, DigitalOcean, and more. You can also leverage the TurboQuant edge network for distributed deployment.
What is TurboQuant and how is it connected?+
TurboQuant (turboquant.network) is a DePIN-style decentralized edge compute network powering the infrastructure behind Freemium.Services. It provides the distributed AI inference, edge caching, and compute resources that enable fast, globally available access to this directory. You can use TurboQuant to host your own self-hosted tools at scale.
How do I submit a tool to the directory?+
Click "Submit Tool" in the navigation bar. Fill out the submission form with the tool's name, URL, category, license type, and a brief description. Our team manually reviews each submission within 48 hours to verify the free tier and add it to the appropriate categories.
Are all tools on this site truly free?+
Yes. Every tool listed is verified to have either: a genuine free tier with no credit card required, an open-source license allowing free self-hosting, or a freemium model with meaningful functionality in the free plan. We clearly label each tool's license type on its listing page.
What is a DePIN network?+
DePIN stands for Decentralized Physical Infrastructure Network. It's a model where physical computing resources (servers, GPUs, storage) are contributed by a distributed network of participants rather than centralized in a single cloud provider's data center. TurboQuant operates a DePIN network specifically optimized for AI inference workloads, offering lower costs and better privacy than traditional cloud providers.

It can help you find the right tools for your use case, compare alternatives, suggest self-hosting strategies, and answer questions about any tool in our directory. It's powered by Claude AI through the TurboQuant network.
